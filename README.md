Adventure Works Bike Sales Analysis Dashboard

Project Overview

This project is an interactive **Power BI Dashboard** created using the **Adventure Works Bike Sales** dataset. The dashboard provides valuable insights into sales performance, customer behavior, product categories, and regional sales distribution through interactive visualizations.

The objective of this project is to analyze business performance and help stakeholders make data-driven decisions.

---

 Dashboard Features

- 📈 Total Sales Overview
- 🛒 Total Orders
- 📦 Total Quantity Sold
- 💰 Average Unit Price
- 📅 Monthly Sales Trend
- 🌍 Country-wise Sales Analysis
- 🚲 Category-wise Sales Performance
- 👥 Customer & Product Analysis
- 🎛️ Interactive Category Filter (Slicer)

---

 Key Insights

- **Total Sales:** **1.68 Million**
- **Total Orders:** **800**
- **Total Quantity Sold:** **2K**
- Sales are distributed across multiple countries including:
  - UK
  - Germany
  - Australia
  - India
  - USA
  - Canada
- Product categories analyzed:
  - Mountain Bikes
  - Road Bikes
  - Touring Bikes
  - Accessories
- Monthly sales trends help identify seasonal demand.
